# 💰 Smart Expense Guard

Smart Expense Guard is a secure, full-stack personal expense tracking web application built using Python, Streamlit, and MySQL (Aiven Cloud).  
It helps users manage daily expenses, analyze spending patterns, and track budgets with OTP-based authentication.

---

## 🚀 Features

- 🔐 OTP-based email login (secure authentication)
- ➕ Add personal expenses
- ✏️ Edit and ❌ delete expenses
- 📊 Interactive dashboard and analytics
- 🔍 Search and filter expenses (category, date, description)
- 💸 Monthly budget tracking with alerts
- 📥 Export expenses to CSV/Excel
- ☁️ Cloud database integration (Aiven MySQL)
- 🔐 Secure credentials using `.env` and Streamlit Secrets

---

## 🛠️ Tech Stack

### Backend
- Python
- MySQL (Aiven Cloud)
- PyMySQL
- SMTP (Gmail OTP)
- dotenv

### Frontend
- Streamlit
- Pandas
- Plotly

### Tools & Platforms
- Git & GitHub
- Streamlit Cloud (Deployment)
- Aiven (Cloud Database)
